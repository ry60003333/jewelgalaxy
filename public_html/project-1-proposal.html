Jewel Galaxy

High Concept

It is like Bejeweled, but better! It allows for a more free form experience, where the player is not limited by a grid.

Genre

The game is a puzzle strategy game.

Platform

Desktop

Story

The player is a god who is playing pool with planets.

Esthetics

Graphics style - Abstract graphics with maybe some "realistic" space backgrounds.

Sound - 8-bit sounds or available free sound assets.

Our game will have scoring sounds, level completion sounds, clicking or action sounds, background music for the levels, and a win and lose sound.

Gameplay

Mechanics

Colored pairs of planets will spawn and move around on the game screen. The main mechanic of the game is collision; when two planets of the same color collide, they annihilate each other. When two planets of different colors collide, they simply bounce off of each other. The goal of the game is eliminate all of the planets.

Control

The main control of the game is clicking (or maybe tapping) on the game screen; This creates an explosion that will push planets away from the origin of the explosion. The closer that a planet is to the origin of the explosion, the further it will move.

Scoring

Scoring will be based off of both the time it takes the player to complete the level, and how many clicks it took the player to beat the level.

Onboarding and Teaching the game

The player will be taught how to play the game by integrating the learning process with the game play. The first level will teach the player the main mechanic and the controls of the game.

Player learning

In order to beat the game, the player needs to learn how to manipulate the planets into bouncing into other planets of the game color by using the force generated by clicking appropriately.
